Detailed Description

Generic icon-based parameter selector.

This component displays multiple selectable icons, each representing one parameter choice.

The selected value is synchronized with a juce::AudioProcessorValueTreeState parameter.

Rendering is delegated through a custom callback allowing waveform, filter, or other icon types.
