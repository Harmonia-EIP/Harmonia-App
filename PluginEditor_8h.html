Detailed Description

Main graphical editor of the Harmonia plugin.

HarmoniaAudioProcessorEditor is the root graphical interface attached to the audio processor.

Responsibilities:

  • Hosting the application UI
  • Managing the root layout
  • Connecting the processor to visual components
  • Displaying the navigation controller

Main UI architecture: HarmoniaAudioProcessorEditor └── AppController ├── WelcomePage ├── LoginPage ├── SignupPage └── MainComponent
