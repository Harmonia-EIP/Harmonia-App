Detailed Description

Root UI controller of the Harmonia application.

This component acts as the central navigation manager responsible for dynamically switching between:

  • Authentication pages
  • Welcome screens
  • Main synthesizer interface

It also manages persistent sessions and backend synchronization during application startup.
