Detailed Description

Real-time oscilloscope waveform display.

OscilloscopeComponent visualizes audio samples received from the synthesizer or DSP pipeline.

The component uses:

  • A circular ring buffer for audio capture
  • A separate draw buffer for rendering safety
  • A JUCE timer for periodic screen refreshes

The audio thread continuously pushes samples into the ring buffer while the GUI thread asynchronously renders the waveform.

Thread Safety:

  • Uses juce::CriticalSection
  • Prevents simultaneous audio/UI access

Rendering Pipeline: pushBuffer() -> ringBuffer

timerCallback() -> drawBuffer

paint() -> waveform rendering

Periodic UI refresh callback.

Called by the JUCE timer at the configured refresh rate.

Responsibilities:

  • Copies ring buffer into draw buffer
  • Reorders wrapped circular data
  • Triggers repaint()

Thread-safe operation is guaranteed using juce::ScopedLock.
