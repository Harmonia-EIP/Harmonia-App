Detailed Description

Handles authentication and session persistence.

This class is responsible for communicating with backend authentication endpoints and managing the user session lifecycle locally.

Covered endpoints:

  • POST /auth/signin
  • POST /auth/signup

Responsibilities:

  • Authenticate users
  • Register new accounts
  • Persist session data to disk
  • Load and clear sessions
  • Synchronize profile data after authentication
Note
Uses BackendManager for API configuration and logging.
