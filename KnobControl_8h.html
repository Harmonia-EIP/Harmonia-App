Detailed Description

Rotary knob UI component linked to an APVTS parameter.

KnobControl provides a custom-styled rotary slider used to manipulate synthesizer parameters.

Features:

  • Rotary interaction
  • Optional bipolar mode
  • Value readout display
  • Fast tweak detection
  • APVTS synchronization
