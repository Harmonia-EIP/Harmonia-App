Detailed Description

Initial landing page allowing the user to choose between sign-in and sign-up.

This component represents the entry point of the authentication flow. It displays:

  • Application title
  • A "Sign in" button
  • A "Sign up" button

The component uses a dedicated dark theme (independent from user preferences) to ensure visual consistency during authentication.

Interaction is handled via a callback function (onChoice), allowing the parent component to react to user selection.

Note
This component does not handle navigation directly; it delegates actions through the callback.
