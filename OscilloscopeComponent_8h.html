Detailed Description

Real-time audio waveform visualization component.

OscilloscopeComponent displays incoming audio signals as a continuously scrolling waveform similar to a hardware oscilloscope.

The component:

  • Receives live audio buffers from the DSP engine
  • Stores samples inside a circular ring buffer
  • Periodically copies data into a rendering buffer
  • Draws the waveform with GPU-friendly JUCE graphics

Features:

  • Real-time waveform rendering
  • Stereo-to-mono signal visualization
  • Adjustable refresh rate
  • Configurable vertical gain
  • Thread-safe audio/UI synchronization
  • Smooth glowing waveform rendering

Architecture: Audio Thread: pushBuffer()

UI Thread: timerCallback() -> copies ringBuffer to drawBuffer -> repaint()

Rendering: paint() -> drawWaveform()
