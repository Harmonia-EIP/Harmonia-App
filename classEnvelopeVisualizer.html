Detailed Description

Visualizes ADSR envelope parameters in real time.

This component listens to APVTS parameter changes and redraws the ADSR envelope whenever a parameter is modified.

The displayed envelope includes:

  • Attack ramp
  • Decay slope
  • Sustain plateau
  • Release curve

The component is purely visual and does not process audio.
