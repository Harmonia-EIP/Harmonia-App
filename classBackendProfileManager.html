Detailed Description

Handles user profile retrieval and updates.

This class communicates with backend profile endpoints to manage user-specific data such as themes and layouts.

Endpoints:

  • GET /profile/me
  • PUT /profile/{id}/theme
  • PUT /profile/{id}/layout

Responsibilities:

  • Fetch user profile
  • Update UI-related preferences
