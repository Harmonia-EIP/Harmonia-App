Detailed Description

Utility namespace used to render waveform and filter icons.

IconRenderer contains helper functions used by selector components to draw synthesizer-related icons.

Supported icons:

  • Oscillator waveforms
  • Filter response shapes

Rendering is vector-based using JUCE Path objects.
