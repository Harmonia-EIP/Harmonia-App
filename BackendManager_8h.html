Detailed Description

Main backend facade used to centralize all backend services.

BackendManager acts as the application's main entry point for all backend-related operations.

Managed services:

  • Authentication
  • User profile management
  • AI preset generation
  • Session persistence
  • User preference synchronization
  • Backend configuration

This architecture simplifies communication between the UI layer and backend services.
