Detailed Description

Real-time safe parameter access structure.

Stores raw atomic<float> pointers retrieved from AudioProcessorValueTreeState.

Benefits:

  • Lock-free DSP access
  • Real-time safe
  • Faster than repeated APVTS lookups
  • Minimal CPU overhead

Used directly inside:

  • Synth voices
  • DSP rendering
  • Modulation systems
