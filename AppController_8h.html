Detailed Description

Main application navigation controller.

AppController manages the complete UI navigation flow of the Harmonia application.

Responsibilities:

  • Restoring saved user sessions
  • Managing authentication screens
  • Displaying the main synthesizer interface
  • Handling logout flow
  • Synchronizing backend session state

Navigation flow: WelcomePage ├── LoginPage ├── SignupPage └── MainComponent
