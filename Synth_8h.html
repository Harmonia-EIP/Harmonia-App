Detailed Description

Core synthesizer voice and sound definitions for Harmonia.

This file contains the main synthesis engine classes used by the Harmonia synthesizer.

Included classes:

  • HarmoniaSound
  • HarmoniaVoice

Main features:

  • Polyphonic synthesizer voices
  • Dual oscillator architecture
  • Multiple waveform generation
  • ADSR amplitude envelope
  • Filter envelope modulation
  • LFO modulation
  • Resonant multimode filtering
  • Velocity sensitivity
  • Distortion processing
  • Noise generation
