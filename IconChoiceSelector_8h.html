Detailed Description

Icon-based selector component for choice parameters.

IconChoiceSelector provides a custom graphical alternative to traditional combo boxes by rendering selectable icons.

Features:

  • Icon rendering callbacks
  • Hover highlighting
  • Active state visualization
  • APVTS parameter synchronization
  • Mouse interaction support
