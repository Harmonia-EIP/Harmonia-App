Detailed Description

Virtual MIDI keyboard UI component for Harmonia.

This component provides a graphical MIDI keyboard interface using JUCE's MidiKeyboardComponent.

It is responsible for:

  • Displaying a playable piano keyboard
  • Synchronizing note states with a shared MidiKeyboardState
  • Providing a visual MIDI input surface for the synthesizer

The component acts as a lightweight wrapper around juce::MidiKeyboardComponent and automatically manages keyboard rendering and resizing behavior.
