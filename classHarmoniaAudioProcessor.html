Detailed Description

Main JUCE audio processor for the Harmonia synthesizer.

This class contains the complete audio engine of the plugin.

Core features:

  • Polyphonic synthesis engine
  • Real-time MIDI synthesis
  • Audio DSP processing
  • Parameter automation support
  • Plugin state persistence
  • Oscilloscope audio monitoring

DSP chain: MIDI Input → Synth Voices → Reverb → Oscilloscope Tap → Audio Output
