Detailed Description

Rotary parameter control component.

This component wraps a JUCE rotary slider and automatically synchronizes it with an AudioProcessorValueTreeState parameter.

Additional features:

  • Velocity-sensitive movement
  • Fast movement callback
  • Custom Harmonia rendering
  • Parameter text display
