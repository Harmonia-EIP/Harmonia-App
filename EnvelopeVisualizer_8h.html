Detailed Description

ADSR envelope visualization component.

EnvelopeVisualizer renders a real-time graphical representation of the synthesizer amplitude envelope.

The component listens to:

  • Attack
  • Decay
  • Sustain
  • Release

parameters from the AudioProcessorValueTreeState.

Features:

  • Real-time updates
  • Animated ADSR curve
  • Gradient envelope fill
  • Sustain and peak markers
  • Harmonia visual styling
